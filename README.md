# Digital Campaign Analytics Dashboard

## Overview

This project analyzes digital marketing campaign performance using Python and Microsoft Power BI.

The workflow includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- KPI Creation
- Power BI Dashboard Development

---

## Dataset

The dataset contains:

- Campaign Name
- Marketing Channel
- Spend
- Revenue
- Clicks
- Impressions
- Conversions
- Device Type
- Target Audience
- Campaign Status
